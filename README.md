# Trading AI

## Overview

This repository contains our engineering project on algorithmic trading using Artificial Intelligence.

The objective is to study short-term Bitcoin (BTC/USDT) price movements and investigate how Machine Learning, Deep Learning, and Reinforcement Learning can be applied to financial markets.

The project is currently in its planning phase, and the implementation details will evolve as the project progresses.
